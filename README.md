# PropEquity-Frontend
Interfaz web interactiva diseñada para la gestión de ventas en casetas inmobiliarias, permitiendo la simulación de créditos bajo el programa Nuevo Crédito Mivivienda y el bono Techo Propio.

Funcionalidades Clave:
Registro dinámico de datos socioeconómicos de clientes y unidades inmobiliarias.

Visualización de cronogramas de pago detallados usando el método francés vencido ordinario (meses de 30 días).

Panel de indicadores financieros en tiempo real (VAN, TIR y TCEA) para asegurar la transparencia de la información.

Sistema de ayuda electrónica integrado para guiar al usuario en cada campo de datos.

Stack: React + Vite + Tailwind CSS.
